"Expletive Deleted" Experiment

Real-time "bleeping and blurring" of web video based on dynamic configuration. Select options below, and the sample video will be scrubbed accordingly. HTML embed code is available to embed player with your configuration.

This should work in Chrome (broken due to Web Audio API bugs in Firefox and Safari). Code by POV Technology Fellow Brian Chirls is available on GitHub. Read more about this experiment on the POV blog.

Thanks to filmmaker Heather Courtney and Quincy Hill Films for allowing us to use a clip from Where Soldiers Come From. Watch the film at http://www.wheresoldierscomefrom.com/dvd/.
